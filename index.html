Understanding Closures & Scope in JavaScript

📘 Intermediate

🏆 +20 XP

📹 Video Lesson · 8:42

What is Scope?

Scope in JavaScript determines where variables can be accessed or referenced. Think of scope as the set of rules that controls how variables are looked up in your code.

JavaScript has three types of scope:

  • Global scope — accessible from anywhere in the program
  • Function scope — accessible only inside the function they're declared in
  • Block scope — variables declared with let or const inside {}
💡 KEY CONCEPT

A closure is created when a function remembers and accesses variables from its outer scope, even after that outer function has finished executing.

Why Do Closures Matter?

Closures allow you to create private variables and factory functions, and are fundamental to understanding how React hooks work under the hood.
